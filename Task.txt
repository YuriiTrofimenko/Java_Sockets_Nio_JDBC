По примеру проекта EclipseLinkDemo создать свою простую БД для СУБД MySql
и консольное приложение, которое будет читать данные из БД,
например, выводить названия групп и студентов к каждой группе.
Для создания единицы персистентности, сущностей и дата-контроллеров использовать EclipseLink
(для этого прочтите инструкцию, как это сделать в Вашей IDE).
